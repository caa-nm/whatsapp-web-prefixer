# whatsapp-web-prefixer

## Descrição

O **WhatsApp Web Prefixer** é uma extensão de navegador, baseada no **Manifest V3**, que automatiza a inserção de prefixos personalizáveis no campo de digitação do WhatsApp Web.

## Contexto e Motivação

Desenvolvida para atender às necessidades internas de comunicação institucional, a extensão não compartilha informações nem coleta dados de telemetria de terceiros.

A ferramenta foi criada para suprir demandas operacionais do **Centro de Agricultura Alternativa (CAA)**, facilitando a identificação de técnicos e setores em atendimentos e comunicações institucionais. Ao adotar uma solução própria, a instituição fortalece sua autonomia tecnológica e reduz a dependência de softwares proprietários ou serviços externos.

---

## Privacidade e Soberania Tecnológica

Alinhada aos princípios de **soberania tecnológica**, a extensão opera com **telemetria zero** e sem qualquer tipo de comunicação externa. Não há rastreamento, coleta de metadados ou envio de dados para servidores remotos. Todas as configurações são armazenadas exclusivamente no navegador, por meio do `chrome.storage.local`, garantindo a privacidade dos dados institucionais.
